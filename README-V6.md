# DroneSense Intelligence Hub — Version 6

This is a **complete Version 6 structural overlay** for the existing static GitHub Pages repository.

## What it adds

- Global market-selection home page for Australia, Canada, European Union, United Kingdom and South Africa.
- UK maintained separately from the EU.
- Australia country hub and interactive state/territory map.
- Correct state routing: every state/territory opens an agency directory first.
- Dedicated profiles for all requested missing police, fire, rural fire, emergency-service and federal/national bodies.
- Critical National Infrastructure separated from state agency navigation.
- Twelve CNI sector pages and a reusable CNI organisation profile template.
- Responsive desktop/mobile shell and shared Version 6 assets.

## Important merge note

The package is an overlay, because the latest full repository ZIP was not available in the active file workspace. Some filenames overlap with existing researched pages. Before replacing an existing researched profile, compare or merge its sourced content into the Version 6 structure.

## Recommended deployment

1. Back up the current repository.
2. Copy the Version 6 files into a new branch.
3. Preserve researched content from existing profiles where a filename overlaps.
4. Test every state map link and agency profile.
5. Commit: `v6: global market architecture and complete Australia agency routing`
6. Deploy the branch to GitHub Pages for review before merging.
