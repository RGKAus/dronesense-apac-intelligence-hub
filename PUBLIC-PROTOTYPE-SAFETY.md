# Public prototype safety note

This repository package is intended as a public-information prototype.

- It contains no individual partner email addresses.
- It contains no customer records, meeting notes or confidential partner information.
- Market and competitor pages use neutral, publicly verifiable wording.
- No authentication, Microsoft tenant connection, database, scraping or automated feed is included.
- Review all new content before publishing and use official source links wherever possible.

A genuine private deployment requires approved access controls outside the static HTML files.
