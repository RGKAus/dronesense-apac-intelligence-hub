Research methodology

This is the platform's working method for taking a topic from "unresearched" to a page carrying a real Confidence Level. Apply it consistently so confidence ratings mean the same thing across every page.
  1. Scope the questionWrite the specific question before searching — "does Agency X operate BVLOS" is answerable; "tell me about drones in policing" is not.
  2. Start at the primary sourceRegulator, legislation register, or the organisation's own site first. Treat news coverage and vendor marketing as leads to verify, not as sources themselves.
  3. Cross-check with a second independent sourceA claim confirmed by one press release is "highly likely" at best. Two independent confirmations moves it toward "confirmed".
  4. Record the access dateRegulatory pages change. Note when you accessed a source, not just when it was published, in the page's Sources section.
  5. Rate confidence honestlyUse Confirmed / Highly likely / Partially verified / Unverified / Historical / Superseded — the same six levels used everywhere on the platform. Don't upgrade a rating to make a page look more finished.
  6. Cross-link before you finishAdd the page to every relevant hub and related-page section so it surfaces from more than one entry point.

AI research workflows

A prompt is a starting point, not an answer. Use the workflow below with any prompt from the Prompt Library.
  1. Run the prompt with search enabledPrompts in this library are written to request sources — if a response has none, ask for them explicitly before trusting any claim.
  2. Open every cited source yourselfAI summaries can misstate regulatory detail. Confirm the specific clause, date or figure on the primary source page before it goes into an intelligence page.
  3. Ask a falsifying follow-up"What would prove this wrong?" or "What's the strongest counter-evidence?" surfaces gaps a single confirming prompt won't.
  4. Convert findings, not prose, into the templateExtract facts into Key Facts / Sources / Confidence Level — don't paste AI-generated summary text directly into Executive Summary.
