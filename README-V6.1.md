# DroneSense Intelligence Hub — Version 6.1 Country Map Portals

This release upgrades every market landing page into a geographic map portal.

## New behaviour

- Australia: clickable states and territories, with federal and national agencies below the map.
- Canada: clickable provinces and territories, with nationwide agencies below the map.
- European Union: clickable EU member states, with EU-wide institutions below the map.
- United Kingdom: clickable UK nations, with nationwide and cross-border agencies below the map.
- South Africa: clickable provinces, with nationwide agencies below the map.
- Every geographic location links to a dedicated public safety profile.
- Every national agency listed below a map links to an individual foundation profile.
- Responsive mobile layout converts map regions into a clean two-column or single-column selector.

## Required next research phase

The Version 6.1 pages establish the navigation, profile structure and working links. Location and agency intelligence should be populated from attributable official and public sources. Existing researched Australian profiles should be preserved when merging.
