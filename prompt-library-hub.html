Regulations

Summarise a regulation, with clause citation

Summarise the current requirements of [regulation name] under [CASR Part 101 / equivalent instrument] in [country]. Cite the specific clause or section number for each requirement, and state the source URL and the date the source was last updated.

Regulations · pairs with the regulation page template

Detect recent regulatory change

What has changed in [regulation or requirement] in [country] in the last 12 months? List each change with the date it took effect, the regulator that issued it, and a link to the primary notice — do not include changes you cannot source.

Compare how [country A] and [country B] regulate [BVLOS operations / RPAS licensing / operator certification]. Present as a table: requirement, [country A] position with source, [country B] position with source, and the practical difference for an operator working across both.

Agencies

Confirm an agency's drone program
